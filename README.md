# Cake Presentation App (แอปพลิเคชันนำเสนอเรื่องเค้ก)

แอปพลิเคชันนี้นำเสนอความรู้เบื้องต้นเกี่ยวกับ "เค้ก" ในรูปแบบสไลด์ที่สวยงาม พัฒนาด้วย Flutter โดยใช้ `PageView` ทำให้สามารถปัดเลื่อน (Swipe) ซ้ายขวาเพื่อดูข้อมูลแต่ละหน้าได้อย่างลื่นไหลเหมือนแอปนำเสนอ (Presentation) หรือ Onboarding 

## ✨ ฟีเจอร์หลัก (Features)

*   **สไลด์นำเสนอ (Slide Presentation):** มีเนื้อหาที่ครอบคลุมถึงความหมาย ประเภท และเสน่ห์ของเค้ก
*   **การนำทางแบบ Custom:** มีปุ่มกด Next / Previous และจุดแสดงตำแหน่งหน้าปัจจุบัน (Dots Indicator) 
*   **ดีไซน์แบบ Responsive:** รองรับการแสดงผลทั้งบนโทรศัพท์มือถือ แท็บเล็ต เดสก์ท็อป และเว็บ (หน้าจอแคบหรือกว้างก็สามารถอ่านได้ง่าย)
*   **UI/UX สบายตา:** เลือกใช้โทนสีพาสเทลและไอคอนที่เข้ากับเนื้อหาเรื่องขนมและเบเกอรี่

## 🚀 เทคโนโลยีที่ใช้ (Tech Stack)

*   **Framework:** [Flutter](https://flutter.dev/) (รองรับ iOS, Android, Web, Desktop)
*   **Language:** Dart
*   **UI Components:** `PageView`, `Stack`, `AnimatedContainer`, `ConstrainedBox`

## 🛠️ วิธีการติดตั้งและรันโปรเจกต์ (Getting Started)

1. **Clone repository หรือแตกไฟล์โปรเจกต์:**
   เข้าไปที่โฟลเดอร์ของโปรเจกต์
   ```bash
   cd cake_presentation_app
   ```

2. **ติดตั้ง Dependencies:**
   ```bash
   flutter pub get
   ```

3. **รันแอปพลิเคชัน:**
   ```bash
   flutter run
   ```

## 📱 โครงสร้างและเนื้อหาภายในแอป (App Structure)

*   **หน้า 1:** ยินดีต้อนรับสู่โลกของเค้ก (บทนำ)
*   **หน้า 2:** เค้กคืออะไร? (อธิบายความหมายและส่วนประกอบ)
*   **หน้า 3:** ประเภทของเค้กที่นิยม (แนะนำสปันจ์เค้ก, บัตเตอร์เค้ก ฯลฯ)
*   **หน้า 4:** ทำไมเราถึงรักเค้ก? (ความสุขของการกินเค้ก)
